## 📖 Sobre este repositório

Este repositório reúne os exercícios, exemplos reescritos e pequenos projetos que venho desenvolvendo enquanto estudo (ou re-estudo) lógica de programação e algoritmos usando JavaScript como linguagem de apoio.

Sou apaixonado por tecnologia desde muito cedo — lá pelos meus 10 anos de idade já me interessava por como as coisas funcionavam por dentro, e essa curiosidade nunca me abandonou. Com o tempo, fui me afastando um pouco da base mais teórica da programação para focar em outras frentes, e hoje sinto que estou "enferrujado" em lógica. Este repositório é minha forma de dar polimento nessa base, revisando os fundamentos com calma, exercício por exercício.

A ideia aqui não é só treinar código — é documentar o processo. Cada capítulo, exercício ou pequeno projeto do livro que eu resolvo é registrado aqui, servindo como um diário de progresso e também como parte da minha apresentação enquanto desenvolvedor no GitHub.

## 💡 Motivação

- Retomar contato com lógica de programação e algoritmos depois de um tempo distante da teoria pura.
- Consolidar fundamentos que sustentam qualquer linguagem ou stack, não só JavaScript.
- Criar um registro público e honesto do meu progresso — incluindo erros, tentativas e refatorações.
- Compartilhar minha trajetória com quem também está começando ou revisando os próprios estudos.

## 📚 Sobre o livro

**Lógica de Programação e Algoritmos com JavaScript**
Autor: Edécio Fernando Iepsen
Editora: Novatec
Edição: 2ª edição, revisada e ampliada

O livro é uma introdução à programação de computadores, com exemplos e exercícios pensados para iniciantes, usando JavaScript como linguagem de ensino.

## 🗂 Estrutura do repositório

```

│   ├── 02-variaveis-e-tipos/

│   ├── 04-estruturas-de-repeticao/
│   ├── 05-vetores-e-matrizes/

│   └── ...
├── exercicios/
│   └── (exercícios propostos pelo livro, resolvidos)

│   └── (projetos finais e desafios maiores)
```

Cada pasta de capítulo contém os exemplos comentados e as soluções dos exercícios propostos, seguindo a ordem do livro.

## 🧭 Como navegar

1. Entre na pasta `capitulos/` para acompanhar o estudo na ordem do livro.
2. Cada exercício resolvido tem um comentário no topo do arquivo explicando o problema proposto.
3. Projetos maiores (final de seções ou do livro) ficam em `projetos/`.
